Python is an intuitive and easy to learn coding language for someone who comes from a beginner coding background. As someone with very little experience coding, I really enjoyed learning Python. I found is made more sense to me than other languages like R. Here are some examples of the ways python is a functional and easy language to use!

If faced with a long list, python makes it very easy to select certain entries. Here is an example.

list1= ["a", "b","c","d"] 
list1[2]
## 'c'
list1 [-1]
## 'd'
list1 [2:]
## ['c', 'd']

This allows a user to select just the position of the entry, without necissarily knowing what the entry is. This is a lot more complicated in other languages like R.

In a given sentence, Python allows a user to count how many times each character shows up. Here is an example:

sentence= "Python is awesome!" 
counts = {}            
for c in sentence:     
    if c in counts:    
        counts[c]+=1   
    else:
        counts[c]=1  
for c in counts:
    print(c, "appears", counts[c],"times")
## ('a', 'appears', 1, 'times')
## (' ', 'appears', 2, 'times')
## ('e', 'appears', 2, 'times')
## ('i', 'appears', 1, 'times')
## ('h', 'appears', 1, 'times')
## ('m', 'appears', 1, 'times')
## ('o', 'appears', 2, 'times')
## ('n', 'appears', 1, 'times')
## ('P', 'appears', 1, 'times')
## ('s', 'appears', 2, 'times')
## ('!', 'appears', 1, 'times')
## ('t', 'appears', 1, 'times')
## ('w', 'appears', 1, 'times')
## ('y', 'appears', 1, 'times')

Returning certain characters from a string is also really easy to do in Python.

import re

string1="I am originally from New York City, which is about 1600 miles away from where I live now, Houston. I moved when I was 8 years old and my sister was 6.5 years old. Today, I go to UT Austin, a 3 hour drive from my hometown. Also, my favorite numbers are 566, 62, 90293, and 12"

re.findall(r"[0-9]+",string1)
## ['1600', '8', '6', '5', '3', '566', '62', '90293', '12']
